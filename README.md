# AppleBee (가제)

> 코드네임: AppleBee | 이전 가제: AgentTycoon

> "돈 버는 다마고치/타이쿤" — AI 에이전트를 게임화하여 누구나 인터넷 부업을 할 수 있게

## 📌 핵심 컨셉

- **타겟**: 초졸도 쓸 수 있게 → 터미널/코드 절대 노출 금지
- **UX**: 심시티/타이쿤 게임처럼 → 사무실, 캐릭터, 레벨업
- **실체**: OpenClaw/Moltbot 기반 AI 에이전트가 실제 부업 수행

## 🎮 게임 요소

| 현실 | 게임 |
|------|------|
| AI 에이전트 | 직원 캐릭터 |
| API 비용 | 직원 월급 |
| 에러/실패 | 직원 울음/시위 |
| 서버 스케일업 | 사무실 확장, 건물 업그레이드 |
| 수익 | 회사 매출 |

## 💰 수익 모델 (BM)

### 1. PPL 피켓 시위 (광고)
- 직원이 "PPL 보셔야 합니다..." 피켓 들고 시위
- 광고를 보면 API 크레딧 충전
- **포인트**: 광고를 "귀여운 이벤트"로 포장 → 거부감 ↓

### 2. 서버비/수수료 (월급)
- AI가 번 돈에서 운영비 차감
- "내 생돈" 나가는 느낌 ↓ (심리적 저항감 최소화)
- 적자 시 → 강제 광고 (유쾌하게)

### 3. 스마트 컨트랙트 (자동 정산)
- 수익의 일정 부분 자동 수수료
- 투명한 정산 → 신뢰도 ↑
- **체인**: Polygon/Base/Arbitrum (L2, 가스비 ↓)

### 4. 캐릭터 스킨/커스터마이징
- 좋아하는 캐릭터로 스킨 변경
- 옷입히기 게임 요소
