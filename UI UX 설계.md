# UI/UX 설계

## 🎯 디자인 원칙

1. **절대 코드 노출 금지** — 터미널, 로그 등 기술적 요소 숨김
2. **게임처럼 느끼게** — 심시티/타이쿤 UX 참고
3. **직관적** — 초졸도 5분 내 이해 가능
4. **중독성** — 레벨업, 보상 시스템으로 리텐션

## 📱 화면 구성

### 1. 메인 화면 (사무실 뷰)

```
┌────────────────────────────────────────┐
│  💰 $127.50        🔔 3      ⚙️        │  ← 헤더
├────────────────────────────────────────┤
│                                        │
│     ┌─────┐   ┌─────┐   ┌─────┐       │
│     │ 📝  │   │ 💻  │   │ 📊  │       │  ← 직원들
│     │마케터│   │개발자│   │분석가│       │     (각자 일하는 중)
│     └─────┘   └─────┘   └─────┘       │
│                                        │
│         [사무실 배경 이미지]             │
│                                        │
│     🪴           ☕           📦       │  ← 장식/가구
│                                        │
├────────────────────────────────────────┤
│  [사무실]  [직원]  [상점]  [통계]  [더보기] │  ← 하단 네비게이션
└────────────────────────────────────────┘
```

**인터랙션**:
- 직원 탭 → 상세 정보 + 작업 현황
- 직원 드래그 → 자리 이동 (커스터마이징)
- 빈 자리 탭 → 새 직원 고용

### 5. 회사 성장 애니메이션

**레벨업 시**:
```
[작은 사무실] → [팀 오피스] → [빌딩] → [본사]

┌───────┐     ┌───────────┐     ┌─────────────┐
│ 🏠    │ →   │ 🏢        │ →   │ 🏙️          │
│ 1인실 │     │ 10인 오피스 │     │ 100인 빌딩   │
└───────┘     └───────────┘     └─────────────┘
```

**트리거**:
- 누적 수익 $100 → 레벨 2 (팀 오피스)
- 누적 수익 $1,000 → 레벨 3 (빌딩)
- 직원 10명 → 레벨 4 (대기업)

## 🎨 비주얼 스타일

**레퍼런스**:
- 심즈 / 심시티 (아이소메트릭)
- 팡야 / 메이플스토리 (캐주얼)
- 카카오프렌즈 (귀여운 캐릭터)
- **핵심 레퍼런스**: ![[ui_reference_01.jpg]]
  - 아이소메트릭 사무실 뷰
  - 귀여운 동물 캐릭터 (고양이)
  - 회의실/협업 느낌
  - 파스텔 톤 컬러

**컬러 팔레트**:
- Primary: #4A90D9 (신뢰감 있는 블루)
- Secondary: #F5A623 (활기찬 오렌지)
- Success: #7ED321 (수익 = 초록)
- Warning: #F5A623 (주의)
- Background: #F8F9FA (밝은 회색)

**폰트**:
- 한글: Pretendard, 나눔스퀘어
- 영문: Inter, Poppins
