# 직원 유형 (에이전트 모듈)

> 각 "직원"은 특정 업무에 특화된 AI 에이전트 모듈

## 🧑‍💼 직원 카테고리

### Tier 1: 기본 직원 (무료/광고)

| 직원 | 역할 | 수익 모델 | 난이도 |
|------|------|----------|--------|
| **인턴 마케터** | 블로그 글 작성, 기본 SEO | 애드센스, 쿠팡파트너스 | ⭐ |
| **댓글 알바** | SNS 댓글 관리, 리뷰 작성 | 건당 수익 | ⭐ |
| **자료 수집가** | 핫딜 정보 크롤링 | 제휴 링크 | ⭐ |

### Tier 2: 숙련 직원 (프리미엄)

| 직원 | 역할 | 수익 모델 | 난이도 |
|------|------|----------|--------|
| **콘텐츠 매니저** | 유튜브 쇼츠, 틱톡 자동화 | 조회수 수익 | ⭐⭐ |
| **이커머스 운영자** | 스마트스토어 관리 | 판매 수수료 | ⭐⭐ |
| **리서처** | 시장 조사, 경쟁사 분석 | 리포트 판매 | ⭐⭐ |

### Tier 3: 전문가 (고급)

| 직원 | 역할 | 수익 모델 | 난이도 |
|------|------|----------|--------|
| **트레이딩 봇** | 암호화폐/주식 모니터링 | 수익 배분 | ⭐⭐⭐ |
| **풀스택 개발자** | 간단한 웹사이트 제작 | 프리랜서 수익 | ⭐⭐⭐ |
| **AI 아티스트** | AI 이미지 생성, 판매 | 스톡 이미지 판매 | ⭐⭐⭐ |

## 📋 직원 상세 스펙

### 1. 인턴 마케터 (Intern Marketer)

**캐릭터 설정**:
- 성격: 열정적, 가끔 실수함
- 대사: "블로그 글 다 썼어요! 검토해주세요~"
- 레벨업 시: "이제 SEO도 할 줄 알아요!"

**기능**:
```yaml
tasks:
  - type: blog_writing
    platforms: [티스토리, 네이버블로그, Wordpress]
    capabilities:
      - 키워드 리서치
      - 글 작성 (1000-2000자)
      - 이미지 삽입
      - 애드센스 최적화
    
  - type: affiliate_marketing
    platforms: [쿠팡파트너스, 알리익스프레스]
    capabilities:
      - 상품 선정
      - 리뷰 작성
      - 링크 삽입

earnings_potential: $50-200/월 (숙련도에 따라)
api_cost: ~$10-30/월
```

**필요 권한**:
- 블로그 계정 로그인
- 쿠팡파트너스 API 키

---

### 2. 콘텐츠 매니저 (Content Manager)

**캐릭터 설정**:
- 성격: 트렌드에 민감, 크리에이티브
- 대사: "이 영상 대박날 것 같아요! 🔥"
- 레벨업 시: "알고리즘 분석도 해드릴게요"

**기능**:
```yaml
tasks:
  - type: shorts_creation
    platforms: [YouTube Shorts, TikTok, Instagram Reels]
    capabilities:
      - 트렌드 분석
      - 스크립트 작성
      - AI 영상 생성 (연동)
      - 자동 업로드
      - 해시태그 최적화
    
  - type: channel_management
    capabilities:
      - 댓글 관리
      - 분석 리포트
      - 업로드 스케줄링

earnings_potential: $100-500/월
api_cost: ~$30-50/월
```

---

### 3. 이커머스 운영자 (E-commerce Operator)

**캐릭터 설정**:
- 성격: 꼼꼼함, 숫자에 강함
- 대사: "오늘 주문 3건 들어왔어요!"
- 레벨업 시: "재고 관리도 자동화할게요"

**기능**:
```yaml
tasks:
  - type: store_management
    platforms: [스마트스토어, 쿠팡마켓플레이스, 11번가]
    capabilities:
      - 상품 등록
      - 가격 모니터링
      - 재고 알림
      - CS 응대 (템플릿)
      - 리뷰 요청
    
  - type: sourcing
    capabilities:
      - 알리/타오바오 상품 리서치
      - 마진 계산
      - 경쟁사 분석

earnings_potential: $200-1000/월
api_cost: ~$20-40/월
```

## 🎨 캐릭터 스킨 시스템

### 기본 스킨 (무료)
- 사무직 기본 복장

### 프리미엄 스킨 (유료/광고)
| 스킨 | 가격 | 설명 |
|------|------|------|
| 정장 | 💎 100 | 프로페셔널 룩 |
| 캐주얼 | 💎 50 | 편한 복장 |
| 할로윈 | 💎 200 | 시즌 한정 |
| 사이버펑크 | 💎 300 | 미래 스타일 |

### 콜라보 스킨 (특별)
- IP 라이선스 협의 필요
- 예: 인기 캐릭터, 게임 캐릭터

## 📈 직원 성장 시스템

```
레벨 1 (인턴)     → 기본 업무만 가능
레벨 5 (사원)     → 효율 +20%, 새 스킬 해금
레벨 10 (대리)    → 자동화 기능 해금
레벨 20 (과장)    → 다른 직원 관리 가능
레벨 50 (팀장)    → 팀 시너지 보너스
```

**경험치 획득**:
- 작업 완료: +10 XP
- 수익 발생: +수익액 × 0.1 XP
- 연속 출근: +일수 × 5 XP
